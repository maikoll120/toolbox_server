# Getting Started

Server-side app for Toolbox Challenge



## Available Scripts ONLY DOCKER

In the project directory, you can run:

### `npm run compose`

## Available Scripts

### `npm start`

### `npm run lint`

### `npm run test`

To test endpoints with Mocha & Chai
